# Akshaya Constructions Website

This is a simple and responsive website built for **Akshaya Constructions** to showcase houses under construction.

## Features

* 🏢 Company header with logo and builder details
* 🏠 Property listings (2BHK, 3BHK, Villa)
* 📍 Clickable Google Maps location for each house
* 📞 Click-to-call contact feature
* 📱 Mobile-friendly design

## Tech Used

* HTML
* CSS

## 🌐 Live Website

👉 https://spurthi-chinnamgari.github.io/Akshaya-Constructions/

## 📂 Project Structure

* `index.html` – Main webpage
* `style.css` – Styling
* Images – Logo and house photos

## 📌 Usage

* View available properties
* Click "View Location" to open in Google Maps
* Click phone number to contact builder

✨ Built for real-world usage to help customers easily explore housing projects.
